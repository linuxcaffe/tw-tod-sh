Taskwarrior Time of Day Shell Script -- help file -- see also: README.md

Usage: ToD [morn|aft|day|eve|night|none|help]

Summary: tw-tod-sh is a taskwarrior Time of Day extension that use tags
    +morn, +aft +day +eve +night to filter some taskwarrior reports 
    (ready and ls, by default) for time-appropriate tasks. 

    Future versions will include integration with the users crontab, changing
    time-of-day setting automatically.
